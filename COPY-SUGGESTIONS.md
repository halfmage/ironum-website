# Copy Analysis & Suggestions: Ironum Homepage (EN + DE)
**Date:** 2026-04-05
**Page Type:** Homepage (B2B AI Services)
**EN Copy Score:** 78/100
**DE Copy Score:** 74/100

## Executive Summary

The English homepage copy is strong — it follows a well-structured PAS (Problem-Agitate-Solve) flow, uses specific numbers (40%, 70%, 100%, 4-8 weeks, 400+ integrations), and addresses a real pain point (GDPR + AI adoption tension). The headline is clear but generic; it could do more to name the audience and differentiate.

The German copy is a solid adaptation, not a translation — it correctly reframes for Mittelstand and uses DSGVO terminology. However, there are several instances where English loanwords break the German flow ("compliant", "Service"), and some sentences are overly long for scanability. The meta description reads like keyword stuffing.

Both versions share a structural strength: the 3-card problem section ("We want to use AI... but we can't risk our data... so nothing happens") is excellent narrative copywriting. This is the single strongest copy element on the page.
